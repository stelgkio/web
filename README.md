## AffilFlow Web (Vite + React)

This frontend is now a Vite React SPA, served locally on port `3001`.

### Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

### Environment

- `VITE_API_URL` (default: `http://localhost:8080`)
- `VITE_APP_ORIGIN` (default: `http://localhost:3001`)

### Routing

Client-side routes are defined in `src/App.tsx` using `react-router-dom`.
